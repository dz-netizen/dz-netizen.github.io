● 脚本能跑了。用法：

全量回测（2998只，和V13报告一样）

python scripts/run_v13.py

先跑100只试试

python scripts/run_v13.py –limit 100

自定义日期 + 保存明细

python scripts/run_v13.py –start 2024-01-01 –output reports/my_report.txt

只看ETF

python scripts/run_v13.py –kind etf –limit 50

运行流程：

  1. 自动寻优：在30只样本上测试5组参数，选最优（min_score_to_enter 在1~2之间搜索）
  2. 全量回测：用最优参数跑全部标的，流式输出进度
  3. 输出：Top 20 摘要 + 统计指标

–output 可以把每笔交易明细保存到文件，格式和 V13_trade_details.txt 一样。
